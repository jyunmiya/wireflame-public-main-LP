Setup Guide

セットアップガイド

このページでは、LRT by 38 APPLICATION を使い始めるまでの手順を説明します。 「難しい」というよりは「手順がある」という感じです。順を追って進めれば、Web 制作の知識がある方であれば問題なく使い始められます。

はじめに

LRT は React で作られたブラウザアプリです。インターネット接続は不要で、手元の PC だけで完結します。 使い方は大きく 2 通りです。

対象読者
HTML や CSS を触ったことがある、FTP でのファイルアップロード経験がある、くらいの方を想定しています。 コードを書く必要はありません。

2通りの使い方

おすすめ

① ローカルで使う

自分の PC でそのまま開く。サーバー契約不要。打ち合わせ持ち込みに最適。

URL共有したい場合

② サーバーに公開

レンタルサーバーにアップロードして URL で共有。クライアントとの画面共有に便利。

① ローカルで使う

ローカルで使うには Node.js のインストールが必要です。一度インストールすれば、以降は start.bat をダブルクリックするだけです。

Node.js のインストール

  1. Node.js 公式サイトにアクセス
    https://nodejs.org を開き、LTS 版（推奨版）をダウンロードします。
  2. インストーラーを実行
    ダウンロードした .msi（Windows）または .pkg（Mac）ファイルを開き、画面の指示に従ってインストールします。すべてデフォルト設定で進めて問題ありません。所要時間は 5 分程度です。
  3. インストールの確認（任意）
    コマンドプロンプト（Windows）またはターミナル（Mac）を開き、以下を入力してバージョンが表示されれば完了です。 
    node -v
一度だけの作業です
Node.js は一度インストールすれば、以後は不要です。別のプロジェクトでも同じ Node.js が使えます。

LRT の起動

  1. フォルダを展開
    ダウンロードした ZIP を展開し、任意の場所に置きます。フォルダ名やパスは何でも構いません。
  2. start.bat をダブルクリック
    フォルダ内の start.bat をダブルクリックします。黒いコマンドウィンドウが開き、しばらくするとブラウザが自動的に起動します。
  3. ブラウザでアプリを開く
    自動で開かない場合は、ブラウザのアドレスバーに http://localhost:3000 と入力してください。
コマンドウィンドウを閉じないでください
黒いウィンドウがサーバーとして動いています。閉じるとアプリも止まります。使い終わったら、そのウィンドウを閉じて終了してください。

終了方法

黒いコマンドウィンドウで Ctrl + C を押すか、ウィンドウ右上の ✕ で閉じてください。

② レンタルサーバーに公開する

URL で共有したい場合は、dist/ フォルダの中身をレンタルサーバーにアップロードします。

  1. dist フォルダを確認
    ダウンロードしたファイルに dist/ フォルダが含まれています。これが公開用のファイル一式です。
  2. FTP ソフトでアップロード
    FileZilla などの FTP ソフトを使い、dist/ フォルダの中身をサーバーのドキュメントルート（public_html/ など）にアップロードします。
  3. URL で確認
    アップロード先の URL をブラウザで開き、アプリが表示されれば完了です。
dist フォルダの中身について
dist/ フォルダには index.html、JS/CSS ファイル、マニュアルページが含まれています。フォルダごとではなく、中身だけをアップロードしてください。

よくある質問

ブラウザが自動で開かない

http://localhost:3000 を手動でアドレスバーに入力してください。コマンドウィンドウが起動してから 5〜10 秒ほど待ってから試してください。

「古い画面が表示される」

ブラウザのキャッシュが残っている場合があります。Ctrl + Shift + R（Mac は Cmd + Shift + R）でハードリロードしてください。

ポート 3000 が使用中と出る

他のアプリが 3000 番ポートを使っている場合があります。他のアプリを終了してから start.bat を再実行してください。

Mac で動かない

start.bat は Windows 用です。Mac の場合はターミナルを開き、フォルダ内で以下を実行してください。

node server.js
